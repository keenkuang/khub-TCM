# khub 4.0 路线图

## 4.0 — 产品打磨（PWA离线完整+UI重设+错误统一）
- PWA Service Worker 缓存策略完善
- Web UI 暗色模式+响应式+骨架屏全覆盖
- 统一错误处理（error_code 全端点覆盖）
- Flutter App 完善（离线缓存+推送通知）

## 4.1 — 生产就绪（安全扫描+CI/CD+监控）
- Docker 安全扫描（Trivy）
- GitHub Actions CI/CD 完善（自动测试+构建+发布）
- Prometheus 告警规则
- 日志聚合（Loki/Grafana）

## 4.2 — AI 深化（可视化 Agent+知识图谱扩增+RAG多模态）
- Agent 可视化拖拽构建器（Web UI）
- 知识图谱自动从文献抽取（NLP pipeline）
- RAG 多模态（图片+表格理解）

## 4.3 — 商业化（支付+运营后台+SaaS）
- 微信支付/支付宝 SDK 集成
- 租户运营管理后台
- SaaS 多租户生产环境验证

## 4.4 — 生态建设（插件市场+SDK+社区）
- 插件市场（在线安装/版本管理）
- Python/JS OpenAPI SDK 生成
- 开发者文档+贡献指南
