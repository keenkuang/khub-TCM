## 核心函数 — `khub/auth.py`

```python
def scope_filter(user: dict | None, resource: str, alias: str = "") -> tuple[str, list]:
    """返回 (where_clause, params) 供 SQL 查询追加。"""
    if not user or user.get("role") in ("admin",) or user.get("via_global_token"):
        return "", []
    role = user.get("role", "")
    uid = user.get("user_id", 0)
    p = alias + "." if alias else ""
    if resource == "patients":
        if role in ("patient", "guardian"):
            return f"{p}id IN (SELECT patient_id FROM consultations WHERE patient_id IN (SELECT id FROM patients WHERE id=?))", [uid]
        if role in ("doctor", "intern"):
            return f"{p}id IN (SELECT DISTINCT patient_id FROM records WHERE patient_id IN (SELECT id FROM patients))", []
        if role in ("nurse",):
            return "", []  # nurse sees all patients for now
    if resource in ("records", "consultations"):
        if role == "patient":
            return f"{p}patient_id=?", [uid]
    if resource == "appointments":
        if role in ("patient", "guardian"):
            return f"{p}patient_id=?", [uid]
        if role == "doctor":
            return f"{p}doctor=?", [user.get("username", "")]
    return "", []
```

## 实施要点
1. 每次列表查询时追加 WHERE 子句
2. 详情查询（get_patient by id）在返回前校验当前用户是否有权访问
3. 后端 store 函数加 `user=None` 可选参数
4. patients 表需要一个 `patient_id` 到 `user_id` 的关联字段（或者直接用 `id` 匹配）
