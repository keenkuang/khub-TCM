## 实施

### 1. 后端补充

`GET /ops/appointments` 分支追加 `qs.get("patient_id")` 参数支持：
```python
pid = qs.get("patient_id", [None])[0]
if pid:
    sql += " AND patient_id=?"
    params.append(int(pid))
```

新增 `GET /clinical/consultations` 端点（查 `consultations` 表）：
```python
if method == "GET" and path == "/clinical/consultations":
    pid = qs.get("patient_id", [None])[0]
    if pid:
        rows = store.conn.execute(
            "SELECT id, patient_id, date, chief_complaint, differentiation, plan FROM consultations WHERE patient_id=? ORDER BY date DESC",
            (int(pid),)).fetchall()
    else:
        rows = store.conn.execute(
            "SELECT id, patient_id, date, chief_complaint, differentiation FROM consultations ORDER BY date DESC LIMIT 50").fetchall()
    return 200, {"consultations": rows}
```

### 2. 小程序端
