# khub 1.1.0 实现计划 — AI Agent 深化

## 核心模块

### Agent 模板市场
- `agent_templates` 表（预置 5 模板：健康助手/辨证助手/预约助手/搜索助手/报表助手）
- `GET /api/agents/templates` — 模板列表
- `POST /api/agents/create-from-template` — 从模板克隆

### Agent 记忆
- `agent_memory` 表（agent_id, key, value, type, created_at）
- `memory.store(agent_id, key, value)` / `memory.recall(agent_id, key)`
- 执行引擎中注入记忆

### 多 Agent 协作
- `agent_pipelines` 表（定义多个 agent 的执行链）
- `pipeline.run(agent_ids, input)` — 顺序执行链
