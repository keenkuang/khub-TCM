# khub 0.9.2 实现计划 — 远程医疗平台

## 核心模块

### 视频问诊
- `khub/telemedicine/signaling.py` — WebRTC 信令（房间创建/加入/交换 SDP）
- `telemedicine_sessions` 表：appointment_id, room_id, status( waiting/in_call/completed), started_at, ended_at
- API：`POST /api/telemedicine/room`（创建房间）、`GET /api/telemedicine/room/{id}`（获取房间信息）

### 电子处方
- `khub/telemedicine/prescriptions.py` — 处方 CRUD
- `prescriptions` 表：consultation_id, doctor_id, patient_id, items(JSON), signature, status(draft/signed/dispensed)
- API：`POST /api/prescriptions`、`GET /api/prescriptions/{id}`
