# khub 1.3.0 实现计划 — 移动端原生 App (Flutter)

## 核心模块

### Flutter 项目结构
- `flutter/` — 完整 Flutter 项目 (Material Design 3)
- REST API 自动生成客户端（基于现有 khub API）
- 离线优先：SQLite 本地缓存 + 同步引擎
- 推送通知：Firebase Cloud Messaging
- 生物识别登录：local_auth

### 页面
| 页面 | 说明 |
|------|------|
| 登录 | 用户名密码 + 生物识别 |
| 首页 | 角色识别后展示对应的仪表盘 |
| 预约 | 预约列表 + 新建预约 |
| 孪生 | 健康摘要 + 时间线 |
| 趋势 | 体质画像 + 证型演变 |
| 患者 | 医生端患者列表 |
| 知识库 | KG 查询（证型→方剂推理） |
