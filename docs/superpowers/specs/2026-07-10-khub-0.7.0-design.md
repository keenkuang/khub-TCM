# khub 0.7.0 设计说明 — AI Copilot 智能助手

## 架构

用户输入 → IntentParser（LLM+离线兜底）→ ToolRegistry → ToolExecutor → 回复

## 核心模块

- `khub/copilot/intents.py`：意图识别（LLM 或正则回退）
- `khub/copilot/tools.py`：工具注册表（8 个预注册工具）
- `khub/copilot/engine.py`：执行引擎（意图→工具→回复）

## 工具清单

search_docs / get_patient / book_appointment / list_courses / query_knowledge_graph / create_notification / run_report / help

## 接口

`POST /api/copilot/chat` — 对话输入，返回工具执行结果
`GET /api/copilot/tools` — 列出可用工具
