# khub 1.0.0 实现计划 — 正式版

## 目标
从 v0.9.4 的 34 版本迭代进入 1.0.0 正式版，核心任务是硬化代码、补齐测试、完善文档、性能验证。

- 所有测试文件加 `pytestmark = pytest.mark.smoke`
- 离线兜底路径加 focused 单元测试
- 确保 `pytest -m smoke` 全绿

### 2. 代码硬化
- `agents/scheduler.py` 实现（spec 承诺未完成）
- `khub/web/icons/khub.png` — PWA 图标（已有占位，验证可用）

### 3. 性能基线
- `pytest -m smoke` 运行时间 < 60s
- 无 flaky 测试

### 4. 文档加固
- CHANGELOG 从 flat 版本列表改为分组分类
- 迁移指南（v0.x → v1.0）

### 5. 版本号
- `khub/__init__.py`: `0.9.4 → 1.0.0`
