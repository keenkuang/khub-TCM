# khub 0.8.2 实现计划 — 数据分析与 ML

## 核心模块

### `khub/analytics.py` — 统计与分析
- `patient_cohorts(store)` — 患者分群（年龄/性别/就诊频次分布）
- `syndrome_efficacy(store)` — 辨证→方剂疗效分析（从 record_struct + followup_adherence 交叉分析）
- `visit_forecast(store, days=30)` — 就诊量预测（基于历史数据简单线性回归）
- `appointment_trends(store, months=6)` — 预约趋势（按月统计 + 同比增长）

### 接口
- `GET /api/analytics/cohorts` — 患者分群
- `GET /api/analytics/efficacy` — 疗效分析
- `GET /api/analytics/forecast` — 就诊预测
- `GET /api/analytics/trends` — 预约趋势
- CLI：`khub analytics-*` 子命令
