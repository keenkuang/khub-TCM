# khub 0.7.2 设计说明 — 统一搜索与知识发现

## 能力

### 跨实体搜索
一次查询搜所有：文档 + 患者 + 课程 + 方剂 + 中药 + 知识图谱证型。

### 混合排序
- FTS5 BM25 分数（文档全文）
- SQL `LIKE` 匹配度（名称/标题）
- 知识图谱关联度（实体间关系路径数）

### 知识发现
- "相关实体"：从知识图谱 `kg_relations` 推荐关联
- 搜索结果分组：按实体类型分组展示

## 实施

### 端点
`GET /api/search?q=感冒&type=all&limit=10`

### 引擎
```python
def unified_search(store, q, type="all", limit=10):
    results = []
    if type in ("all", "docs"): results += search_docs(store, q, limit)
    if type in ("all", "patients"): results += search_patients(store, q, limit)
    if type in ("all", "courses"): results += search_courses(store, q, limit)
    if type in ("all", "herbs"): results += search_herbs(store, q, limit)
    if type in ("all", "formulas"): results += search_formulas(store, q, limit)
    results.sort(key=lambda r: r.get("score", 0), reverse=True)
    return results[:limit]
```
