# khub 0.8.3 实现计划 — 集成中心

## 核心模块

### 企业微信/钉钉机器人通知
- `khub/integrations/bot.py` — 通用机器人消息推送（WeChat Work / DingTalk）
- 环境变量配置 webhook URL
- `notifications.create()` 事件自动推送到机器人

### 集成状态看板
- `GET /api/integrations/status` — 所有已配置集成的状态
- 环境变量检测：WECHAT_WEBHOOK / DINGTALK_WEBHOOK / KHUB_LLM_URL 等

### Webhook 管理 UI
- 0.6.0 已有 Webhook 订阅端点，补充 Web UI 管理面板
