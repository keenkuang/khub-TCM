# khub 0.9.0 实现计划 — AI Agent 平台

## 核心模块

### `khub/agents/store.py` — Agent 定义 CRUD
- `agent_definitions` 表：name, description, system_prompt, tools(JSON), schedule, active
- CRUD：`create_agent` / `list_agents` / `get_agent` / `update_agent`

### `khub/agents/engine.py` — Agent 执行引擎
- `run(agent_id, user_input)` → 加载 agent 定义 → 拼接系统 prompt → 工具选择 → 执行 → 格式化回复
- 复用 `khub/copilot/tools.py` 的 8 个预注册工具
- 支持工具链自动组合（一个工具输出作为下一个工具输入）

### `khub/agents/scheduler.py` — 定时 Agent
- 基于 `khub/scheduler.py`，定时触发 agent 执行
- `agent_schedules` 表：agent_id, cron_expression, last_run, status
- CLI：`agent-schedule` 子命令
