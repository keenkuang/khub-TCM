# khub 0.6.2 设计说明 — 高级 BI 与报表

## 三大能力

### 报表引擎
- `report_templates` 表：存储报表定义（名称、SQL 查询、图表类型、格式）
- `report_jobs` 表：报表运行记录（触发时间、状态、输出路径）
- `khub/reports.py`：报表模板 CRUD + 执行 SQL + 格式化输出

### 数据看板
- 扩展 `/stats` 端点，增加关键经营指标
- Web UI 仪表盘页面（卡片布局：预约数/营收/活跃患者/课程报名）

### 导出
- `GET /api/reports/{id}/export?format=csv|xlsx|pdf` — 导出报表数据
- PDF 通过 HTML→wkhtmltopdf 或纯文本格式
- CSV 直接输出（text/csv）

## 实施

### 表
```sql
CREATE TABLE report_templates (id INTEGER PK, name TEXT, description TEXT, query TEXT, chart_type TEXT, format TEXT DEFAULT 'table', config TEXT, created_at TEXT);
CREATE TABLE report_jobs (id INTEGER PK, template_id INTEGER, status TEXT, output TEXT, error TEXT, created_at TEXT, completed_at TEXT);
```
