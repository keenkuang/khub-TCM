# khub 2.3 实现计划 — 垂直行业产品化（中医诊所 SaaS）

## 核心模块

### 计费与收费
- `khub/clinic/billing.py` — 费用项 + 收费 + 发票
- `billings` 表：appointment_id, items(JSON), total, paid, method(cash/wechat/alipay), status

### 药房管理
- `khub/clinic/pharmacy.py` — 库存 + 发药 + 库存预警

### 诊所看板
- 扩展 `/stats` 加入诊所经营指标
