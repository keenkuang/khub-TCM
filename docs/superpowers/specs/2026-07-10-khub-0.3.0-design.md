# v0.3.0 设计说明 — 多用户鉴权基础

## 功能
- `users` 表 + `auth_tokens` 表（PBKDF2-SHA256 密码哈希）
- `POST /auth/login` / `logout` / `GET /auth/me`
- API 鉴权从单 KHUB_API_TOKEN 切换为按用户 JWT
- Web 登录页 + CLI `khub login/logout/whoami`
- 向后兼容：KHUB_API_TOKEN 仍可用

## 文件
- `khub/auth.py` — hash_password / verify_password / issue_token / validate_token
- `khub/web/login.html`

## 原则
- 零外部依赖（stdlib hashlib + secrets）
