# khub 0.5.0 设计说明 — 中医知识图谱与推理引擎

## 数据模型

5 张业务表（`kg_herbs` / `kg_formulas` / `kg_syndromes` / `kg_methods` / `kg_relations`），`kg_relations` 为通用关系边支撑推理。

## 推理范式

```
证型 → (indicates) → 治法 → (indicates) → 方剂 → (contains) → 中药 → (tropism) → 归经
                                     ↘                                          ↙
                                  (contraindicates) ———— 禁忌传播
```

## 输出

6 个模块 + 种子数据 + 推理引擎 + API/CLI + 测试。
