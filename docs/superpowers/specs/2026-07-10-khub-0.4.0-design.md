# khub 0.4.0 设计说明 — 临床智能增强（四线合一）

基于已有临床数据底座（twin_v2 / extract / followup / RAG），新增智能分析层。

## 四线合一

| 线 | 模块 | 描述 |
|----|------|------|
| 1 | `analysis.py` | 临床知识图谱：证型→方剂关联矩阵、体质演变分析 |
| 2 | `visualize.py` | 孪生可视化数据：健康趋势、体质画像推断 |
| 3 | `diagnosis.py` | AI 辅助辨证：知识库 + LLM 推荐 + 配伍禁忌检查 |
| 4 | `tracking.py` | 疗效追踪：就诊频次 + 随访依从性 → 疗效评估 |

所有能力走 `get_provider()`，`NoOpProvider` 退回离线知识库。
