## 实施

### 彩色输出（ANSI 无依赖）
```python
class Color:
    HEADER = '\033[95m'; BLUE = '\033[94m'; GREEN = '\033[92m'
    YELLOW = '\033[93m'; RED = '\033[91m'; END = '\033[0m'; BOLD = '\033[1m'
    @staticmethod
    def green(s): return f"{Color.GREEN}{s}{Color.END}"
    @staticmethod
    def red(s): return f"{Color.RED}{s}{Color.END}"
    @staticmethod
    def yellow(s): return f"{Color.YELLOW}{s}{Color.END}"
    @staticmethod
    def blue(s): return f"{Color.BLUE}{s}{Color.END}"
    @staticmethod
    def bold(s): return f"{Color.BOLD}{s}{Color.END}"
```

### 结构化错误
api.py 中的错误返回统一加 `error_code` 字段：
```python
return 401, {"error": "unauthorized", "error_code": "AUTH_001", "message": "请提供有效的认证令牌"}
```

### 补全脚本
`khub completion --bash` / `--zsh` 生成自动补全脚本。
